1. Instalación de node.js

Es necesario descargar este archivo de instalación del sitio oficial Node.js pulsaremos al botón Download Node.js y se nos iniciará la descarga.

Una vez descargado, lo abriremos y ejecutaremos el archvio .msi comenzará el proceso de instalación.

Hacemos doble clic en el archivo de instalación y ejecutamos. El instalador nos pedirá que aceptemos el acuerdo de licencia de Node.js. Marcaremos la casilla de Aceptar y le daremos a Siguiente.

Seleccionaremos el destino donde deseamos instalar Node.js. De normal la ruta por defecto que no sales es esta C:\Program Files\nodejs\ pero si queremos poder cambiar de ruta sin problema. Una vez cambaido le daremos de nuevo a Siguiente.

Lo siguiente que se nos muestra es las opciones de configuración personalizada. Si quieres una instalación estándar con las características por defecto de Node.js, le damos a Siguiente. Sino es así, puedes darle a los botones desplegables y más tarde darle a Siguiente una vez elegido lo que más te convenga.

Node.js te ofrece opciones para instalar herramientas para módulos nativos. Si estás interesado en ellas, haz clic en la casilla para marcar tus preferencias, o haz clic en Siguiente para seguir con la opción predeterminada.

Por último, le daremos al botón de instalar y en unos segundos o minutos nos mostrará que se ha instalado con éxito. Le daremos al botón Finalizar para así, poder cerrar el instalador de Node.js


Si queremos comprobar la versión que nos hemos instalado, solo tenemos que abrir nuestra cmd y escribir npm --version

2. Abrir PowerShell como administrador

Iremos a la búsqueda de Windows y escribiremos PowerShell, clicaremos ya sea Click derecho - Ejecutar como administrador o ya sea como se muestra en la imagen.

Escribiremos la siguiente sentencia Get-ExecutionPolicy, esto nos sirve para saber la politica de ejecución que tenemos actualmente en nuestro sistema.

Una vez dado al Enter, nos saldrá el siguiente mensaje Restricted esto nos quiere decir que no se permite ejecutar ningún script, pero no os preocupeis que esto tiene su solución.

Para ello, vamos a escribir Set-ExecutionPolicy Unrestricted, con esto cambiaremos la politica de ejecución para que así nos permita la ejecución de todos los scripts, sin ninguna restricción.

Después de pulsar Enter, se nos muestra una información que nos pregunta si estamos de acuerdo en cambiar esta politica, nosotros le daremos a la tecla S.

Nos volverá a preguntar si queremos cambiar la directiva de ejecución. A nosotros nos interesa teclear a Sí a todo a la tecla O.


3. Abrir cmd

Abrimos el cmd para realizar lo siguiente:

Escribiremos por consola la instalación de la version 17 de Angular que es la siguiente sentencia npm install -g @angular/cli@17

Se nos mostrará una barrita girando, esto significa que se nos está instalando.

Acabado el proceso de la instalación de Angular, se nos muestra los paquetes instalados y los segundos que ha podido tardar en realizarlo.

Además, si observais, ya nos indica que usemos un conmando para saber donde se encuentra la instalación.

Lo ejecutamos y vemos donde se encuentra, en mi caso en mi Usuario de Windows.


Para la creación de nuestra aplicación, deberemos de crear antes un directorio, por ejemplo misProyectos.

Podemos comprobar que se ha creado correctamente con el comando dir.

Nos meteremos dentro del directorio creado, para poder crear desde ahí nuestra aplicación.


Para poder crear nuestra aplicacion, deberemos de escribir por consola ng new mi-aplicacion.

Antes de la intalación nos preguntará si estamos de acuerdo de compartir nuestros datos con Google. Esto es totalmente opción, podemos darle a la tecla y de aceptar o n de negar.

Nos preguntan cual hoja de estilos usamos, en mi caso elegí css porque de esta forma nos permite seprar la estructura del contenido de la presentación.

De nuevo nos preguntará si queremos habilitar el server SSR, esto lo que hace es renderizar nuestra aplicación del lado del servidor. Como esto nos interesa, le daremos a la tecla y de que estamos de acuerdo.

Ahora se mostrará la creación de la aplicación:

Una vez finalizada la creación de la aplicación, se nos debe de mostrar algo así:

IMPORTANTE

Si por alún casual la instalación no se ha completado como esperabas, te recomiendo borrar todo y volver a instarlo

Nos metemos dentro de la aplicación con el comando cd y el nombre asignado a la aplicación.

Una vez dentro, ejecutamos el siguiente comando para abrirlo ng serve -o.

Nos vuelven a preguntar si estamos de acuerdo de compartir nuestros datos con Google. Esto es totalmente opción, podemos darle a la tecla y de aceptar o n de negar.

Se nos cargará el proceso.

Hasta que se nos abre el navegador con el contenido de la página, esto es totalmente modificable y lo podemos hacer a nuestro gusto.

4. Escribir el código en VS (Visual Studio)

Abrimos Visual Studio.

Seleccionaremos la carpeta donde se encuentra nuestris proyectos, haciendo lo siguiente File - Open Folder...

Nos dirigiremos a la ubicación donde tenemos el directorio que hemos creado, en mi caso misProyectos.

Abierto el directorio, se nos muestra diferentes carpetas. Las que más nos interesa son SRC - APP y el HTML app.component.html y app.component.css


En app.component.html borraremos todo su contenido y colocaremos el codigo que nosotros queramos.
